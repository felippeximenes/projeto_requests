# Programming Languages Used in Repositories of Major Companies

### Description
This project was developed as part of the Data Analysis course at Alura. The goal is to collect, analyze, and store data on the most used programming languages in the repositories of major companies like Amazon, Netflix, and Spotify.

### Language Used
Python: Used to make requests to the GitHub API, manipulate the obtained data, and save the results in CSV files.

### Tools

- Requests: Library used to make HTTP requests to the GitHub API.
- Pandas: Library used for data manipulation and analysis.
- Base64: Used for encoding the files before uploading them to GitHub.

### Main Function
The main function of this project is to extract information about the public repositories of certain companies, such as the repository names and the programming languages used in them. This data is then stored in CSV files, which are subsequently uploaded to a new repository on GitHub.

### Summary of Steps

1- Data Collection: Using the GitHub API, the repositories of companies like Amazon, Netflix, and Spotify are listed.
2- Analysis and Processing: The repository names and programming languages are extracted and organized into a DataFrame.
3- Data Storage: The processed data is saved in CSV files.
4- Upload to GitHub: The CSV files are uploaded to a new repository on GitHub, facilitating access and visualization of the information.

#### Motivation

This project was an excellent opportunity to apply concepts learned in the Data Analysis course at Alura, combining programming skills with data analysis, and providing a practical view of how large companies use different programming languages.

## Linguagens Utilizadas nos Repositórios de Grandes Empresas

### Descrição
Este projeto foi desenvolvido como parte do curso de Análise de Dados da Alura. O objetivo é coletar, analisar e armazenar dados sobre as linguagens de programação mais utilizadas em repositórios de grandes empresas como Amazon, Netflix e Spotify.

### Linguagem Utilizada
- **Python**: Usada para fazer requisições à API do GitHub, manipular os dados obtidos, e salvar os resultados em arquivos CSV.

### Ferramentas
- **Requests**: Biblioteca utilizada para fazer as requisições HTTP à API do GitHub.
- **Pandas**: Biblioteca utilizada para manipulação e análise dos dados.
- **Base64**: Utilizada para codificação dos arquivos antes do upload para o GitHub.

### Função Principal
A principal função deste projeto é extrair informações sobre os repositórios públicos de determinadas empresas, como os nomes dos repositórios e as linguagens de programação utilizadas neles. Esses dados são então armazenados em arquivos CSV, que são posteriormente carregados em um novo repositório no GitHub.

### Resumo das Etapas
1. **Coleta de Dados**: Utilizando a API do GitHub, os repositórios de empresas como Amazon, Netflix e Spotify são listados.
2. **Análise e Processamento**: Os nomes dos repositórios e as linguagens de programação são extraídos e organizados em um DataFrame.
3. **Armazenamento de Dados**: Os dados processados são salvos em arquivos CSV.
4. **Upload no GitHub**: Os arquivos CSV são carregados em um novo repositório no GitHub, facilitando o acesso e a visualização das informações.

### Motivação
Este projeto foi uma excelente oportunidade de aplicar conceitos aprendidos no curso de Análise de Dados da Alura, combinando habilidades de programação com análise de dados, e oferecendo uma visão prática de como grandes empresas utilizam diferentes linguagens de programação.
